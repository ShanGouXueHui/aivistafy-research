# 一、核心升级原则

## 1）市场交易的不是事件本身，而是事件对宏观变量的影响
错误直觉：
强硬表态 -> 市场暴跌

正确模型：
地缘事件
-> 能源运输是否受阻
-> 油价是否失控
-> 通胀与利率路径是否改变
-> 风险资产折现率是否抬升
-> 股票再定价

---

# 二、模型1：地缘政治传导模型

## 核心链条
地缘冲突
-> 霍尔木兹 / 能源运输
-> 油价
-> 通胀预期
-> 利率路径
-> 风险偏好
-> 股市

## 结论
若能源链未失控，市场通常不会系统性崩盘；
若停火/缓和窗口出现，风险溢价会快速回吐。

# 三、模型2：盘口反直觉模型

## 关键现象
- 主动卖单占比高
- 股价却上涨

## 解释
存在更强被动买盘，常见于：
- 机构承接
- 大资金吸筹
- 有计划的回补仓位

## 结论
盘口方向 ≠ 资金方向；
主动卖多但价格上涨，不能简单理解为弱势。

---

# 四、模型3：反弹 vs 反转

## 修复行情特征
- 指数反弹
- 个股跟涨
- 未放量突破关键均线
- 更多是风险溢价回补

## 反转行情特征
- 放量上涨
- 主动买盘主导
- 突破关键均线
- 板块与个股同时强化

## 当前经验
本轮更接近：
- 风险修复
而不是：
- 趋势反转

---

# 五、模型4：A股 vs 港股 定价差模型

## A股
- 情绪与交易属性更强
- 修复日更容易先弹
- 资金活跃度更高

## 港股
- 更偏机构定价
- 更偏基本面和风险偏好折现
- 通常更便宜，赔率更高

## 当前经验
- A股更强：适合观察短线交易性
- 港股更便宜：适合观察中期赔率

---

# 六、模型5：资金分层行为

| 资金类型 | 典型行为 | 对价格的作用 |
|--------|---------|-------------|
| 机构 | 承接 / 定方向 | 决定中期路径 |
| 游资 | 放大波动 / 追逐催化 | 决定短期弹性 |
| 散户 | 恐慌卖出 / 追涨 | 常为反向指标 |
| 量化 | 做市 / 套利 / 提供流动性 | 不主导趋势 |

---

# 七、模型6：预测 -> 回测 -> 修正

每次预测之后，必须做：

1. 对比实际走势
2. 判断方向是否正确
3. 判断区间是否命中
4. 分析误差来源
5. 修正模型

不得只做预测，不做回测。

---

# 八、当前固定结论

1. 单一新闻标题不能直接映射为股灾
2. 必须优先看能源、油价、利率路径
3. 风险修复日：
   - 指数先涨
   - 高弹性板块修复更快
   - 个股是否反转仍需看自身结构
4. A股与港股需要分开建模

---

# 九、一句话总结

> 从“看消息”升级为“看传导链条”；从“看涨跌”升级为“看资金与定价结构”。
