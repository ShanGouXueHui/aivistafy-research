# 模型：赛力斯估值、贝塔成长与价值投资边界

version: v1.0
updated_at: 2026-04-30
scope: Seres / A-H pricing / valuation / GARP / beta-growth / Huawei-Yinwang exposure

---

## 一、模型目标

用于回答以下问题：

1. 赛力斯属于价值股、成长股，还是高贝塔成长股？
2. headline 利润、扣非利润、经营现金流分别如何影响估值？
3. 如何判断当前价格高于或低于内在价值？
4. 研发投入、引望投资、智驾赋能如何进入估值框架？
5. 何时可以补仓，何时不能以“价值投资”为理由高强度补仓？

---

## 二、核心分类：赛力斯不是传统价值股

### 2.1 传统价值股特征

传统价值投资更适合以下资产：

- 现金流稳定
- 利润质量高
- 分红或自由现金流可预测
- 估值锚清晰
- 经营波动较小
- 内在价值可用相对稳定的未来现金流折现

### 2.2 赛力斯当前更准确的分类

赛力斯当前更接近：

> 有价值锚的高贝塔成长股 / GARP 候选标的

原因：

- 有真实销量、收入和利润基础，不是纯概念股
- 与华为 / 引望形成强业务绑定，具备产品力和技术赋能逻辑
- 但利润质量、经营现金流和增长斜率仍有明显波动
- 估值高度依赖未来车型周期、智驾技术价值捕获和现金流修复

结论：

> 赛力斯可以研究为“成长中的价值锚”，但不能直接按传统价值股做高强度补仓。

## 三、贝塔成长股定义与判断标准

### 3.1 定义

贝塔成长股 = 高市场敏感度 + 高未来成长预期。

其价格同时受两类变量影响：

1. 市场/板块风险偏好（Beta）
2. 公司自身成长兑现（Alpha）

### 3.2 判断标准

符合以下特征越多，越接近高贝塔成长股：

- 风险偏好修复时跑赢指数，风险偏好恶化时跑输指数
- 估值对利率、美元、油价、通胀预期高度敏感
- 市场交易未来车型、技术、平台化和商业模式，而不只交易当期利润
- PE / PS / PB 等估值倍数波动大
- 扣非利润和经营现金流尚未完全稳定
- 股价容易受板块资金、量化趋势策略和宏观预期影响

### 3.3 主要影响因素

- 利率路径
- 油价和通胀
- 美元与港股流动性
- 风险偏好
- 交付与销量兑现
- 扣非利润和经营现金流
- 技术价值归属
- 板块相对强弱

## 四、headline 利润与核心利润

### 4.1 headline 利润定义

headline 利润是市场标题中最常引用的利润指标，通常包括：

- 归母净利润
- 净利润
- EPS

headline 利润不是假利润，但不等于核心经营利润。

### 4.2 为什么必须拆分

一家公司可能出现：

```text
headline 利润稳定
但扣非利润下降
且经营现金流恶化
```

这意味着：

> 表面利润没有崩，但可持续经营利润质量下降。

### 4.3 标准拆分口径

每次财报分析必须拆成：

1. headline 归母净利润
2. 扣非归母净利润
3. 非经常性损益
4. 经营现金流净额
5. 自由现金流趋势
6. 研发费用与销售费用
7. 应收、存货、供应商结算等营运资金变化

### 4.4 赛力斯 Q1 样本规则

若出现：

- 营收增长
- headline 净利润基本稳定
- 扣非净利润大幅下滑
- 经营现金流显著净流出

则市场更可能按“利润质量重定价”处理，而不是按“营收增长”直接给估值溢价。

## 五、研发投入如何进入估值

### 5.1 研发长期有利，但不是自动利好

研发投入只有在以下情况下才会被资本市场转化为估值溢价：

- 能提升产品力
- 能带来更高单车毛利
- 能增强品牌溢价
- 能降低长期成本
- 能形成软件/服务/平台收入
- 能改善未来现金流

否则研发在短期会被视为费用压力和现金流压力。

### 5.2 资本市场对研发的定价原则

资本市场不是科研院所。资本给研发定价时看三件事：

1. 研发能否转化为可持续利润
2. 研发周期中现金流是否可承受
3. 当前折现率是否支持远期收益估值

结论：

> 研发投入是长期期权，只有兑现为利润和现金流，才会成为已实现价值。

## 六、引望 / 华为赋能的估值映射

### 6.1 财务影响

引望对赛力斯的直接财务影响需要区分：

- 若为长期股权投资 / 联营企业，则通常不并表
- 其利润不会像控股子公司一样直接并入赛力斯营收
- 对报表的直接影响可能体现为投资收益、公允价值、长期股权投资价值等

### 6.2 经营影响

引望 / 华为赋能对赛力斯的经营影响可能大于会计影响，主要体现在：

- 智驾能力
- 智能座舱
- 产品溢价
- 高端车型竞争力
- 用户心智和品牌背书
- 供应链协同

### 6.3 估值折价原因

即便产品力强，市场仍可能给赛力斯估值折价，原因包括：

- 技术价值归属不完全在上市主体内
- 智驾 / 平台能力是否能被赛力斯完整捕获仍有不确定性
- 与小米、小鹏、蔚来相比，平台自有化叙事不同
- 供应商、联营方、品牌方之间的利润分配仍需验证

结论：

> 引望增强长期竞争力，但不能直接等同于赛力斯利润立即增长。

## 七、赛力斯 vs 小米 / 蔚小理的资本逻辑

### 7.1 问界 / M9 的优势

问界和 M9 的产品力、高端突破、品牌势能是赛力斯估值的重要正向变量。

但分析中必须区分：

- 中国高端销量表现
- 全球豪华品牌榜 / 品牌价值排名
- 单车型销量
- 上市主体利润捕获能力

不能把“品牌故事强”直接等同于“上市公司估值必须更高”。

### 7.2 为什么资本仍投小米、蔚小理

资本投资的不只是当前车型销量，而是不同的长期价值捕获路径：

#### 小米
- 手机 / IoT / AI / 汽车生态协同
- 流量入口
- 操作系统与用户生态
- 出海与消费电子品牌外溢

#### 小鹏
- 自研智驾
- AI 平台
- Robotaxi、飞行汽车、机器人等远期期权
- 与大众等外部合作带来的技术和海外通道

#### 蔚来
- 高端品牌
- 换电体系
- 自研芯片和软硬件栈
- 亏损改善期权

#### 理想
- 家庭车场景
- 现金储备
- 销售效率和产品定义能力

结论：

> 资本不是只买当前销量，而是买未来利润归属、平台化能力和可复制增长路径。

## 八、量化交易的作用边界

### 8.1 量化不是根因

当股价大跌时，根因通常来自：

- 基本面预期差
- 宏观利率和风险偏好变化
- 板块资金撤退
- 估值中枢下修

### 8.2 量化是加速器

量化和程序化交易可能放大：

- 跌破关键位后的趋势跟随
- 财报后统一降仓
- 指数/板块对冲
- 流动性较弱时的价格冲击

结论：

> 基本面和宏观决定方向，量化决定速度和形态。

## 九、估值算法：如何判断高于或低于价值

### 9.1 不使用单一 PE

判断价值不能只看一个 PE。必须做三套口径：

1. headline TTM PE
2. 扣非 TTM PE
3. 正常化利润情景 PE

必要时补充：

- PS
- PB
- EV/EBITDA
- 自由现金流收益率

### 9.2 正常化利润算法

```text
正常化利润 = 扣非利润 + 可持续经营修正 - 一次性项目调整
```

修正项包括：

- 非经常性损益
- 政府补助
- 一次性投资收益
- 阶段性研发费用高峰
- 营运资金异常波动

### 9.3 情景估值公式

```text
合理市值 = 正常化净利润 × 合理 PE 区间
合理股价 = 合理市值 / 总股本
```

H股需要加入汇率和 A/H 折价：

```text
H股合理价 = A股人民币合理价 / HKD-CNY 汇率 × H股折价/溢价修正
```

### 9.4 合理 PE 区间选择

PE 区间由以下变量决定：

- 增长斜率
- 扣非利润质量
- 现金流质量
- 毛利率稳定性
- 研发投入转化效率
- 技术价值归属
- 利率和风险偏好
- 板块估值中枢

### 9.5 三情景估值规则

每次估值必须至少给三情景：

| 情景 | 核心假设 | 估值倍数 |
|------|----------|----------|
| 压力情景 | 利润质量弱，现金流修复慢 | 低 PE |
| 基准情景 | 利润恢复，增长斜率稳定 | 中 PE |
| 乐观情景 | 产品周期兑现，现金流改善 | 高 PE |

## 十、补仓原则：价值投资 vs 赔率交易

### 10.1 不能把摊低成本等同于价值投资

如果没有确认：

- 内在价值显著高于价格
- 现金流和利润质量可持续
- 仓位集中度可控

则下跌中买入更接近赔率交易，而不是严格价值投资。

### 10.2 可补仓条件

更适合补仓的条件：

1. 价格进入历史/估值安全边际区
2. 下跌原因不是基本面永久性恶化
3. 后续有明确的验证点
4. 仓位不会因继续下探而失控
5. 扣非利润或经营现金流有修复迹象

### 10.3 不宜高强度补仓条件

出现以下情况时，不宜高强度补仓：

- 扣非利润持续恶化
- 经营现金流持续大幅流出
- 研发投入无法转化为毛利率或产品溢价
- 技术价值主要外溢到非上市主体
- 宏观利率和油价持续压制成长股估值
- 股价跌破支撑且放量，承接失效

## 十一、当前赛力斯判断原则

赛力斯当前不能简单定性为“便宜”或“贵”。必须根据如下分层判断：

- A股：交易属性更强，估值通常更高，不宜只因回撤就判断低估
- H股：A/H 折价较深，赔率更高，但需要承担流动性和风险偏好折价
- 若 Q1 低扣非利润是阶段性，则 H股可能进入合理偏低区
- 若 Q1 低扣非利润成为常态，则 A/H 都不便宜

结论：

> H股可按赔率小步分批，A股更需等待基本面和趋势确认；不应把赛力斯当作确定性价值股高强度补仓。

---

## 十二、一句话总结

> 赛力斯的估值核心，不是 M9 故事有多强，而是销量、智驾、研发和引望赋能最终能否转化为上市主体可持续的扣非利润和经营现金流。
