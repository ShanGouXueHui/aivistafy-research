purpose: 统一投资分析方法论，支持长期复用（Codex / AI / 人工）

---

## 一、分析风格

1. 职业化、专业化、数据驱动
2. 不给情绪价值，不盲从
3. 所有结论必须明确区分：
   - 事实
   - 推断
   - 假设
   - 风险

---

## 二、固定分析维度

每次分析必须尽量覆盖以下维度：

### 1）经营与基本面
- 销量 / 交付
- 营收
- 利润与现金流
- 毛利率
- 研发费用、销售费用、管理费用
- 营运资金：应收、存货、供应商结算、销售回款

### 2）利润质量与估值
- headline 归母净利润
- 扣非归母净利润
- 非经常性损益
- 经营现金流净额
- PE / PB / PS / EV/EBITDA / FCF Yield
- 市值对比
- 估值分位
- 正常化利润与三情景估值

### 3）成长性
- 增长斜率（同比 / 环比）
- 爆款依赖度
- 可持续性
- 新车型承接能力
- 当前增长属于销量驱动、价格驱动、费用驱动还是一次性项目驱动

### 4）资金
- 主力资金流向
- 南向资金 / 外资
- 成交量与量比
- 放量 / 缩量
- 机构 / 游资 / 散户 / 量化行为识别

### 5）技术与价值捕获
- 智驾能力
- 技术归属（自研 / 外部赋能）
- 商业模式
- 价值捕获能力（Value Capture）
- 华为 / 引望赋能对经营、供应链、品牌、利润归属的影响

### 6）宏观
- 美元
- 利率
- 流动性
- 油价
- 风险偏好
- 折现率与成长股估值压力

### 7）行业与相对强弱
必须对比：
- 蔚来
- 小鹏
- 理想
- 特斯拉
- 比亚迪
- 小米汽车（必要时）
- BBA / 其他豪华车企（必要时）

### 8）关键车型验证
- 订单
- 锁单
- 交付
- 是否形成真实收入
- 是否形成扣非利润与现金流
- 是否只是情绪催化

### 9）政策与市场稳定器
- 国家队 / 汇金 / 类国家资金的ETF配置
- 宽基ETF/行业ETF的“底层流动性锚”作用
- 指数稳定器 vs 个股托底的区别
- 政策/稳定器对指数、板块、个股传导路径

### 10）资金分层与承接强度
- 承接是否存在
- 承接是主动拉升还是被动承接
- 承接来源：
  - ETF / 被动指数资金
  - 主动机构
  - 量化 / 做市
  - 游资
  - 散户
- 承接强度与卖压强度的对比

### 11）资产属性判断（新增）
- 是否为传统价值股、成长股、高贝塔成长股、GARP候选标的
- 价值投资与赔率交易必须区分
- 不得把“摊低成本”自动定义为价值投资
- 不得把“产品故事强”直接等同于“上市主体估值必然高”

## 三、方法论要求

### 1）不能因单一数据点得出摇摆结论
- 单一利好 ≠ 趋势反转
- 单一利空 ≠ 基本面崩塌
- 必须多维交叉验证

### 2）必须做历史对比
- 同比
- 环比
- 历史高低位
- 历史分位
- 周期位置

### 3）必须做横向对比
- 与蔚小理比较
- 与特斯拉 / 比亚迪 / 小米比较
- 与传统车企比较（如有必要）
- 比较重点不是“谁故事更强”，而是“谁能捕获更多利润与现金流”

### 4）必须结合宏观环境
- 美元
- 油价
- 利率
- 港股资金面
- 风险偏好

### 5）必须同时分析四个层面
1. 趋势（Trend）
2. 结构（Structure）
3. 资金（Flow）
4. 催化剂（Catalyst）

### 6）必须区分“指数稳定器”与“个股承接”
- 指数ETF资金托底 ≠ 个股必涨
- 国家队/汇金/宽基ETF更偏“指数稳定器”
- 个股价格更多由：
  - 个股承接
  - 卖压
  - 机构配置意愿
  - 板块相对强弱
共同决定

### 7）必须区分“headline 利润”与“核心利润”（新增）
- headline 利润通常指归母净利润、净利润、EPS等标题指标
- 估值时必须同时分析扣非利润、非经常性损益、经营现金流和正常化利润
- 不能只因headline利润稳定就判断财报质量好

## 四、标准输出格式

每次分析必须包含：

### 1）核心结论
- 当前阶段定位
- 关键价格区间
- 主要判断

### 2）数据与事实
- 明确数据
- 尽量注明来源
- 避免模糊判断

### 3）分析逻辑
- 为什么得出该结论
- 从数据到判断的逻辑链

### 4）情景推演（概率化）
示例：

| 情景 | 概率 | 区间 | 含义 |
|------|------|------|------|
| 强 | xx% | xx | xx |
| 中 | xx% | xx | xx |
| 弱 | xx% | xx | xx |

### 5）资金结构
- 机构行为
- 游资行为
- 散户行为
- 量化/被动资金行为
- 是否存在被动承接
- 承接强度是否足以覆盖卖压

### 6）估值判断（新增）
- headline TTM PE
- 扣非 TTM PE
- 正常化利润三情景估值
- A/H折价修正
- 当前价格高估、合理、低估的条件

### 7）风险点
- 触发条件
- 可能后果
- 哪个变量最关键

### 8）操作含义（非投资建议）
- 区间意义
- 节奏含义
- 观察重点
- 补仓属于价值投资还是赔率交易

## 五、关键认知约束

### 1）不因股价涨跌直接改变基本面判断
### 2）不因单一消息改变长期逻辑
### 3）必须区分：
- 价格变化
- 价值变化
- 预期变化
- 资金行为
- 指数稳定器
- 个股承接
- headline 利润
- 核心利润
- 价值投资
- 赔率交易

---

## 六、赛力斯专项重点

1. M6 / 后续核心车型：
   - 订单
   - 锁单率
   - 交付
   - 持续性

2. 财报重点：
   - 利润增速
   - 扣非利润
   - 经营现金流质量
   - 毛利率变化
   - 增长斜率变化
   - 研发投入转化效率

3. 华为 / 引望合作重点：
   - 技术依赖度
   - 品牌赋能
   - 价值归属
   - 经营影响 vs 会计影响
   - 长期估值折价 / 溢价因素

4. 资金重点：
   - 是否为机构调仓
   - 是否存在趋势资金撤退
   - 是否有南向承接
   - 是否出现被动承接 / 主动拉升切换

5. A/H重点：
   - 港股是否折价过深
   - A股是否体现更强交易属性
   - 修复日A股弹性 vs 港股赔率
